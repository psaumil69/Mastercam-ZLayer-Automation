# Installation Guide

## Supported Versions
This NET-Hook supports Mastercam 2022 and newer.

## Installation Steps
1. Download the DLL and FT files from the GitHub Release.
2. Place both files in your Mastercam chooks folder:C:\Program Files\Mastercam 202X\chooks\
3. Right‑click SplitLayers.dll → Properties → check **Unblock** (recommended).
4. Restart Mastercam.
5. Open the NET-Hook from:
   Add‑Ins → NET‑Hook menu

## Troubleshooting
- NET-Hook does not appear:
  • Ensure DLL is unblocked  
  • Verify Mastercam version matches the DLL  
  • Confirm files are in the correct chooks folder  
  • Restart Mastercam after copying the files  

- Error loading:
  • Ensure the .FT file is placed next to the DLL  

## Uninstallation
Delete the DLL and FT files from the chooks folder and restart Mastercam.
